# handWritten-digit-recognition-based-on-OpenCV
我的第一个项目，原本是软件工程课程设计
还有许多不足需要改进orz

基于OpenCV3.1的简单的手写数字（MNIST库）识别系统

目前版本使用的算法为SVM，特征为8 * 8全部像素与28 * 28（原图）直方图

实现功能1：
    在控制台画板（opencv实现）上用鼠标进行书写并识别（支持多数字识别）
